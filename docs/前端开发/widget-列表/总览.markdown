#### widget包含什么

* **axios**：发起请求。
* **image**: 上传图片和浏览图片。
* **camera**：照相机相关。
* **notice**：通知相关，toast，modal，alert等等
* **storage**：存储相关：存到客户端本地。
* **router**：路由相关。
* **geo**：获取当前位置。
* **nav**：原生导航条设置相关。
* **events**：事件相关，页面和页面通信，js 和 native 通信。
* **pay**：支付相关。
* **tools**：工具相关，如收起键盘等。
* **share**：分享相关。
* **font**：字体设置相关。
