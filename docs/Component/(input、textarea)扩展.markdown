# &lt;input&gt;、&lt;textarea&gt; 扩展

### 扩展样式

修改光标颜色：tint-color

示例

```js
<template>
    <div>   
        <text>这是用路由打开的一个新页面</text>
        <input style="width:750px; height:80px; top:100px; font-size: 60px; tint-color: red;" type="text" placeholder="Input Text" value=""/>
    </div>
</template>
```
